MESSAGE D'ACCUEIL

Ce modèle de Page d'accueil est idéal pour introduire le contenu de votre site Web. Pratiquement la totalité de son contenu peut être personnalisée, y compris les images, le texte et les liens. Vous pouvez choisir de conserver les graphiques proposés ou les remplacer par les vôtres.

Le texte de cette page est destiné à vous aider à dynamiser la conception de votre site en vous conseillant le type de contenu approprié, sans brider votre imagination. Il en va de même pour le texte des liens : n'hésitez pas à le modifier en fonction de vos besoins spécifiques. Ce modèle de Page d'accueil peut servir de base à plusieurs pages de votre site. Cependant, vous pouvez également choisir parmi les modèles de page Texte, Catalogue, Produits ou Calendrier. Montrez-vous créatif et n'hésitez pas à combiner les différents modèles afin de rendre votre site aussi attrayant que possible. Pour toute question relative à la modification de ces pages à l'aide de Contribute, consultez le fichier Lisez-moi récupéré lors du téléchargement ou le Système d'aide du logiciel. Amusez-vous bien et réalisez un site Web formidable !
